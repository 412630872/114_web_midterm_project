# 專案名稱 - 淡江美食網

## 組員 
- 葉建緯
- 葉宥良

## 專案簡介
- 這是淡江美食推薦網站，網站的作用是介紹淡江大學附近不同類型的餐廳，可以給瀏覽網頁的使用者提供選擇，網站內附有餐廳位置、資訊讓使用者可以方便查閱，最後還有表單填寫，給使用者評論餐廳意見。

## 使用技術
- HTML（語意化標記）
- CSS（響應式與深色主題）
- Bootstrap 5（排版與元件）
- JavaScript（DOM 操作、驗證邏輯、localStorage）
- Constraint Validation API（setCustomValidity() 等自訂驗證）

## 功能特色
- 表單即時驗證
- 星級評分
- 深色/淺色切換
- 表單資料記憶
- 防止重複送出
- 送出確認反饋
